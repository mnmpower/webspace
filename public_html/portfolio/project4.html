Project 4.0 - Monitoring van de veiligheid rond een bewegende last

Opdrachtgever: 3 IT

Klant: Aperam

20 Januari 2020 - 28 Februari 2020

Tijdens de lessen van "Project 4.0” in 3 ITF heb ik als deel van een multidisciplinair team gewerkt aan een project om de veiligheid in het staalbedrijf Aperam te verbeteren. Dit project moest een totaaloplossing worden die zowel de veiligheid verbeterde als de nodige overtredingen bijhield.

De opdracht

PROJECT UITLEGTOEVOEGEN

Analysefase

Voor dit project heb ik samen met mijn team een lange analysefase achter de rug. Deze duurde 8 weken waarin we elke week één namiddag samen kwamen. Iedereen had dan opzoekwerk gedaan over 1 bepaald aspect. We zaten samen en bespraken dit uitvoerig. Dit schreef ik dan neer in een analyse document. Na de analysefase kregen we 4 weken de tijd om dit project effectief uit te werken. Ik heb dus ook 2 aparte GitHub repositories aangemaakt om de projecten om een goede manier mee bij te houden.

Realisatiefase

Mijn bijdrage aan dit project was het maken van de applicatie. Als frontend framework heb ik gekozen voor Angular. De bijhorende API heb ik geschreven in asp.NET. De applicatie zelf bestaat uit 2 delen. Het eerste deel is en dashboard dat ergens in het bedrijf getoond kan worden waar alle overtredingen van het personeel op worden getoond. Het tweede deel is de beheerderskant waarop alles nauwgezet opgevolgd kan worden. Zo heeft de beheerder een live overzicht van de gevarenzonen waar hij de werknemers op kan volgen. Zodra een werknemer een overtreding maakt, wordt er automatisch een melding gemaakt en opgeslagen.

Omdat het project een week voor onze deadline al af was heb ik nog een extra taak op mij genomen. Deze viel volledig buiten mijn vakgebied. Ik heb een opstelling gemaakt op een raspberry pi met een ultrasoon sensor. Deze detecteerde wanneer er iemand zich onder onze testopstelling bevond. Als dit het geval was, sprak de pi de API aan en schreef een melding weg in de database. Deze opstelling was enkel om een betaalbaar voorbeeld te hebben op onze projectstand op de beurs.
