Project 4.0 - Monitoring van de veiligheid rond een bewegende last

Opdrachtgever: 3 IT

Klant: Aperam

20 Januari 2020 - 28 Februari 2020

Tijdens de lessen van "Project 4.0” in 3 ITF heb ik als deel van een multidisciplinair team gewerkt aan een project om de veiligheid in het staalbedrijf Aperam te verbeteren. Dit project moest een totaaloplossing worden die zowel de veiligheid verbeterde als de nodige overtredingen bijhield.

Binnen het bedrijf Aperam is er een bewegende brugkraan. Dit is een enorme kraan die over het grootste deel van de werkvloer kan bewegen. Deze kraan kan een zware ballast dragen en is nodig voor de werking van het bedrijf. Het is uiteraard niet veilig om hier onderdoor te lopen maar dit gebeurt toch. Omdat dit niet mag volgens de veiligheidsvoorschriften is er gevraagd of we een oplossing konden bedenken om de werkvloer te monitoren en deze dan ook te realiseren.
Voor dit project besteedde mijn team acht weken aan de analysefase. Elke week kregen we één namiddag om hieraan te werken. Iedereen had dan opzoekwerk gedaan over één bepaald aspect. We zaten dan samen om dit uitvoerig te bespreken. Dit schreef ik dan neer in een analysedocument. Na de analysefase kregen we vier weken tijd om dit project effectief uit te werken. Ik heb dus ook twee aparte GitHub repositories aangemaakt om de projecten op een goede manier bij te houden.
Mijn bijdrage aan dit project was het maken van de applicatie. Als frontend framework heb ik gekozen voor Angular. De bijhorende API heb ik geschreven in asp.NET. De applicatie zelf bestaat uit twee delen. Het eerste deel is een dashboard met alle overtredingen van het personeel. Het tweede deel is de beheerderskant waarop alles nauwgezet opgevolgd kan worden. Zo heeft de beheerder een live overzicht van de gevarenzonen waar hij de werknemers kan volgen en een pagina waar alle overtreding op staan.
