Project 4.0 - Monitoring van de veiligheid rond een bewegende last

Opdrachtgever: 3 IT

Klant: Aperam

20 Januari 2020 - 28 Februari 2020

Tijdens de lessen van "Project 4.0” in 3 ITF heb ik als deel van een multidisciplinair team gewerkt aan een project om de veiligheid in het staalbedrijf Aperam te verbeteren. Dit project moest een totaaloplossing worden die zowel de veiligheid verbeterde als de nodige overtredingen bijhield.

De opdracht

PROJECT UITLEGTOEVOEGEN

Analysefase

Voor dit project besteedde mijn team acht weken aan de analysefase. Elke week kregen we één namiddag om hieraan te werken. Iedereen had dan opzoekwerk gedaan over één bepaald aspect. We zaten dan samen om dit uitvoerig te bespreken. Dit schreef ik dan neer in een analysedocument. Na de analysefase kregen we vier weken tijd om dit project effectief uit te werken. Ik heb dus ook twee aparte GitHub repositories aangemaakt om de projecten op een goede manier bij te houden.

Realisatiefase

Mijn bijdrage aan dit project was het maken van de applicatie. Als frontend framework heb ik gekozen voor Angular. De bijhorende API heb ik geschreven in asp.NET. De applicatie zelf bestaat uit twee delen. Het eerste deel is een dashboard met alle overtredingen van het personeel dat ergens in het bedrijf getoond kan worden. Het tweede deel is de beheerderskant waarop alles nauwgezet opgevolgd kan worden. Zo heeft de beheerder een live overzicht van de gevarenzonen waar hij de werknemers kan volgen. Zodra een werknemer een overtreding begaat, wordt er automatisch een melding gemaakt en opgeslagen.

Omdat het project één week voor onze deadline al af was heb ik nog een extra taak op mij genomen. Deze viel volledig buiten mijn vakgebied. Ik heb een opstelling gemaakt op een Raspberry Pi met een ultrasoonsensor. Deze detecteert wanneer er iemand zich onder onze testopstelling bevond. Als dit het geval was, sprak de Raspberry Pi de API aan en schreef een melding weg in de database. Deze opstelling was enkel om een betaalbaar voorbeeld te hebben voor onze projectstand op de beurs.
